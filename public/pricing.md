# Pricing Structure – CC Bins

CC Bins is an enterprise-grade payment routing and credit card Bank Identification Number (BIN) intelligence tool. We provide free web interface queries, developer API sandbox access, and commercial database snapshots.

## 1. Web Lookup Interface (Free)
*   **Cost:** $0.00 / month
*   **Volume Limit:** Up to 1,000 bulk BIN query lookups per request
*   **Capabilities:** Full database search, issuer filtering, card brand detection (Visa, Mastercard, Amex, Discover, etc.), funding category filtering, and Luhn algorithm validations.

## 2. API Developer Sandbox (Free Beta)
*   **Cost:** $0.00 (Beta Sandbox)
*   **Volume Limit:** Up to 60 queries per minute (sliding window rate limit)
*   **Capabilities:** REST API endpoint, JSON response format, edge-cached delivery, CORS support.

## 3. Premium Database Download (Commercial)
*   **Cost:** $149.00 (One-Time Purchase)
*   **Format:** Sanitized CSV and SQLite database files
*   **Rows:** 376,900+ verified BIN/IIN ranges
*   **Updates:** Weekly updates provided via private download link for 12 months
*   **License:** Commercial use permitted (unlimited database copies within your organization)
